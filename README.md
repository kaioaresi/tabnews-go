# tabnews-go

Recriando tabnews em golang

## Install golang-migration

```bash
wget https://github.com/golang-migrate/migrate/releases/download/v4.18.2/migrate.linux-amd64.deb
sudo dpkg -i migrate.linux-amd64.deb
```
